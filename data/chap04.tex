\chapter{基于威胁势场与多头注意力机制的多无人艇协同围捕博弈策略}
\label{cha:hunting}

\section{引言}
相较于单艇追逃博弈, 多无人艇协同围捕博弈不仅要求追击者逼近逃逸者, 还需要多艇之
间在空间上形成包围态势, 共同限制逃逸者的机动空间直至将其捕获. 在含障碍物的海洋
环境中, 协同围捕面临以下三方面挑战: 其一, 现有研究大多未给出严格的围捕成功数学
判据, 导致算法评价依据模糊; 其二, 围捕过程呈现明显的阶段性特征 (接近、成型、保持),
单一的奖励函数难以同时兼顾各阶段的学习需求, 常导致稀疏奖励与训练收敛缓慢; 其三,
现有工作多止步于数值仿真, 缺乏真实水面环境下的物理平台验证, 算法的工程可实现性
仍有待考察.

针对上述问题, 本章围绕多无人艇协同围捕博弈策略的设计与验证, 开展以下四个方面的工
作: (i) 在六自由度欠驱动运动学模型的基础上, 给出围捕成功的显式数学判据, 解决现有
研究中评价标准模糊的问题; (ii) 提出能量波峰 (Energy Crest, EC) 与能量波谷 (Energy
Trough, ET) 概念, 将威胁势场转化为反映接近-成型-保持三阶段的密集奖励机制; (iii) 在
MAPPO 算法的 Critic 网络中引入多头注意力机制, 形成 MHD-MAPPO 算法, 以自适应
地聚焦关键特征、抑制冗余观测; (iv) \textbf{搭建多无人艇物理实验平台, 在真实水面环境
中完成 2 对 1 和 3 对 1 协同围捕实物实验}, 从仿真到实物全面验证所提算法. 本章内容
在作者已发表的文献~\cite{wang2025cooperativehunting} 基础上整理扩展而成.

\subsection{围捕成功数学判据}
\label{subsec:hunt_criterion}
定义以逃逸者为中心的包围区域 $\Omega(t)$ 为
\begin{equation}
\Omega(t) = \left\{ (x,y)\ \big|\ \| (x,y) - \mathbf{p}_m^C(t) \| \le d_{CA} \pm \sigma \right\},
\label{eq:besieged_area}
\end{equation}
其中 $\mathbf{p}_m^C(t)$ 为包围区域中心, $d_{CA}$ 为围捕判据半径, $\sigma$ 是允许包围
半径动态调节的容差因子. 进一步, 设 $R_e(t)$ 为逃逸者的可达活动区域, 则协同围捕成
功的严格数学判据定义为
\begin{equation}
\left\{
\begin{aligned}
\mathbf{p}_E(t) &\in \Omega(t), \quad &\forall t \ge t_0, \\
R_e(t) &\subseteq \Omega(t), \quad &\forall t \ge t_0,
\end{aligned}
\right.
\label{eq:hunt_cond}
\end{equation}
并要求追击者通过策略最小化逃逸者在包围区域边界上的最大距离, 即
\begin{equation}
\min_{\pi}\ \max_{(x,y)\in R_e(t)}\ d\big((x,y),\ \partial\Omega\big).
\label{eq:min_max}
\end{equation}

\subsection{Critic 网络中的多头注意力机制}
\label{subsec:mha_critic}
% 【待填充】
% 详细介绍: Multi-Head Attention 结构; Q/K/V 投影; 多头并行; 输出拼接再线性映射.
% 在 Critic 中的输入: 全局状态由 N 个追击者观测拼接; MHA 允许网络自适应聚焦最关键的
% agent-对-agent 关联, 相比传统单头自注意力结构仅能学习单一相关性子空间, 多头机制
% 在多智能体场景下能更充分地捕捉不同语义层次的特征依赖.

\subsubsection{注意力计算}
多头注意力的计算公式为
\begin{equation}
\mathrm{Attention}(Q,K,V) = \mathrm{softmax}\!\left(\frac{QK^T}{\sqrt{d_k}}\right) V,
\end{equation}
\begin{equation}
\mathrm{head}_i = \mathrm{Attention}(QW^Q_i,\ KW^K_i,\ VW^V_i),
\end{equation}
\begin{equation}
\mathrm{MHA}(Q,K,V) = \mathrm{Concat}(\mathrm{head}_1,\ldots,\mathrm{head}_h)W^O,
\end{equation}
其中 $h$ 为注意力头数, $W^Q_i,\ W^K_i,\ W^V_i,\ W^O$ 为可训练参数矩阵.

\begin{algorithm}[!htbp]
  \caption{MHD-MAPPO 多无人艇协同围捕算法}
  \label{algo:mhd_mappo}
  \DontPrintSemicolon
  \SetKwInOut{Input}{输入}
  \SetKwInOut{Output}{输出}
  \Input{Actor 参数 $\{\theta_i\}_{i=1}^N$, Critic 参数 $\phi$, 经验池 $\mathcal{D}$}
  \Output{训练完成的策略 $\{\pi_{\theta_i}\}_{i=1}^N$}
  初始化 Actor 与 MHA 增强的 Critic 网络参数\;
  \For{迭代 $k = 1,2,\ldots,K$}{
    \For{并行环境 $e = 1,\ldots,E_p$}{
      每个追击者 $i$ 使用 $\pi_{\theta_i}$ 采集轨迹\;
      计算基于三阶段 TPF 机制的密集奖励 $r_t$\;
    }
    利用 MHA-Critic 计算全局优势 $A_t$\;
    \For{epoch $= 1,\ldots,E$}{
      以裁剪目标更新各 Actor 参数 $\theta_i$\;
      更新 Critic 参数 $\phi$\;
    }
  }
\end{algorithm}

\section{本章小结}
本章研究了复杂环境下多无人艇协同围捕博弈问题. 首先, 建立了基于六自由度非线性运
动学模型的协同围捕问题, 并给出了围捕成功的显式数学判据; 其次, 提出了能量波峰与能
量波谷概念, 将威胁势场转化为接近-成型-保持三阶段的奖励机制; 再次, 在 MAPPO 算法
的 Critic 网络中引入多头注意力模块, 构建了 MHD-MAPPO 算法; 随后, 通过多场景仿
真实验验证了所提算法在训练效率、协同围捕成功率与泛化能力上的优势; 最后, 搭建了
多无人艇物理实验平台, 在真实水面环境下完成 2 对 1 与 3 对 1 协同围捕实物实验, 系统
验证了算法从仿真到实物迁移的工程可行性, 是本章研究的关键创新点之一.
